Example games

After you're finished with these examples and you want to look at more code, try the example games. In particular:

  • Nebula Alpha - a minimal but complete game using the JGame base classes

    NebulaAlpha.java

  • Chain Reaction - a small but complete game using the JGame base classes

    ChainReaction.java

  • Space Run I, II, III - Space Run I and II demonstrate how StdGame can be used, with resp. defaults and with custom settings. Space Run III illustrates scrolling and wraparound.

    SpaceRun.java | SpaceRunII.java | SpaceRunIII.java

  • Matrix Miner - a more complex game with tiles and using StdGame and Std objects.

    MatrixMiner.java
